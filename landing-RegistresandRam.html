Pestillo AND OR (AND-OR LATCH)

Representa el circuito base para almacenar datos, esta compuesto por una compuerta OR, una AND y una NOT.

Este circuito tiene la capacidad de recordar un BIT de información. Es llamado LATCH porque registra un valor particular y se mantiene de esa manera.La acción de poner datos en la memoria se llama escritura, mientras que sacar los datos se conoce como lectura.

Ahora, tener dos datos de entrada sigue siendo complejo y el siguiente paso es convertir esta entrada de data y reset en un solo dato de entrada, y hará falta una entrada para habilitar la escritura o bloquear esta acción, este se conoce como línea de habilitación de escritura. Con unas compuertas adicionales se construye este circuito y se conoce como pestillo con cierre (GATED LATCH).
