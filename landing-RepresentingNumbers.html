ASCII y UNICODE

Francis Bacon, famoso escritor ingles, usaba secuencias de 5 bits para codificar las 26 letras del alfabeto inglés y enviar mensajes secretos en el año 1600, pero solo letras minúsculas. Para almacenar textos entra en juego ASCII (Código americano de estándares para el intercambio de información). Inventado en 1963, ASCII era un código de 7 bits, capaz de almacenar 127 valores diferentes.

ASCII fue un estándar tan pionero, fue ampliamente utilizado por diferentes computadores construidas por diferentes empresas capaces de compartir información, esta habilidad se llegó a conocer como interoperabilidad. Aunque ASCII tenía una limitación, solo fue diseñado para soportar el idioma inglés.

Los japoneses estaban familiarizados con este problema de codificación que tenía un nombre especial para ellos: Mojibake que significa texto revuelto.De esta manera nació UNICODE, un formato para dominarlos a todos. La versión más común de UNICODE usa 16 bits.
